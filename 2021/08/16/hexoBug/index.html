(一)Cannot read property ‘groupBy’ of undefined

Cannot read property ‘groupBy’ of undefined

2.处理

网上大多数是说lodash函数引起的，我安装了：
npm install lodash
hexo g时还是报上述错误

最后屏蔽掉文件archive.jade如下内容，即可：
文件：themes/maupassant/layout/archive.jade
使用//进行屏蔽
